DSP Tools: Post-campaign Heatmaps
Статический веб-сервис для создания тепловых карт по итогам открутки pDOOH.
Что умеет
загружает Excel-отчет по кампании;
строит карту фактических OTS / контактов;
показывает выходы и потраченные средства;
сохраняет созданные карты в браузере;
открывает клиентскую карту без доступа к внутреннему интерфейсу;
дает фильтры по городу и формату;
переключает ореолы по контактам, выходам и средствам;
считает рейтинг поверхностей 1-5 внутри каждого города.
Важно
Excel обрабатывается локально в браузере пользователя и не отправляется на сервер.
Созданные карты в текущей версии хранятся локально в браузере.
Текущая версия
v0.3 - функциональный прототип портала: загрузка отчета, сводка по данным, сохранение карт и клиентский просмотр.
