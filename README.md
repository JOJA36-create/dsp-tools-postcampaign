# DSP Tools v0.4 Server Prototype

Локальный серверный прототип портала DSP Tools.

## Что внутри

- загрузка Excel-отчета в браузере;
- сохранение созданной карты на локальном сервере;
- список карт из серверного хранилища;
- постоянная клиентская ссылка вида `/map/:id`;
- отдельный клиентский просмотр карты без доступа к порталу;
- удаление карт из серверного списка.

## Как запустить

```bash
node server.js
```

После запуска откройте:

```text
http://127.0.0.1:8787
```

Данные сохраняются в `data/maps.json`.

## Важно

Это локальный прототип без настоящей авторизации и без PostgreSQL.
Следующий этап - заменить JSON-хранилище на базу данных и добавить реальный вход для коллег.
