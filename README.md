# DSP Tools v0.4.2 Server Prototype

Локальный серверный прототип портала DSP Tools для post-campaign pDOOH heatmaps.

## Что внутри

- загрузка Excel-отчета в браузере;
- создание тепловой карты по фактическим OTS / контактам;
- группировка ореолов по OTS, выходам и средствам;
- рейтинг поверхностей 1-5 внутри каждого города;
- сохранение созданных карт в `data/maps.json`;
- список созданных карт в портале;
- клиентская ссылка вида `/map/:id` без доступа к порталу;
- удаление карт из серверного списка.

## Запуск локально

Нужен Node.js 18 или новее.

```powershell
cd C:\Users\evgka\Documents\pDOOH\dsp-tools-server
npm start
```

После запуска откройте:

```text
http://127.0.0.1:8787
```

## Переменные окружения

```text
PORT=8787
HOST=0.0.0.0
```

Локально можно ничего не задавать. Для хостинга важно, чтобы сервер слушал `0.0.0.0`.

## Данные

Карты временно сохраняются в:

```text
data/maps.json
```

Это рабочий прототип. Следующий этап для продакшена: нормальная база данных, настоящая авторизация и роли пользователей.
