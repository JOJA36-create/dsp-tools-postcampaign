# DSP Tools: Post-campaign Heatmap Generator

Статический веб-сервис для создания тепловой карты по итогам открутки pDOOH.
Это версия сервиса DSP Tools для отчетов по фактическим OTS.

## Как пользоваться

1. Откройте сайт.
2. Загрузите Excel-отчет по кампании.
3. Нажмите "Собрать карту".
4. Откройте карту или скачайте готовый HTML.

## Что внутри

- портальный интерфейс DSP Tools;
- создание новой карты из Excel-отчета;
- локальная библиотека сохраненных карт в браузере;
- фильтры по городу и формату;
- ореолы по фактическим OTS / контактам;
- рейтинг поверхностей 1-5 внутри каждого города;
- переключение подложек карты;
- локальная обработка Excel в браузере.

## Важно

Файл Excel обрабатывается в браузере пользователя и не отправляется на сервер.
Сервис использует CDN-библиотеки XLSX и Leaflet, а также онлайн-картографические тайлы.

## Публикация на GitHub Pages

1. Создайте публичный репозиторий, например `dsp-tools-postcampaign`.
2. Загрузите в корень репозитория файлы `index.html` и `README.md`.
3. Откройте `Settings` -> `Pages`.
4. В блоке `Build and deployment` выберите `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. После сохранения GitHub выдаст ссылку вида:
   `https://username.github.io/dsp-tools-postcampaign/`

## Версии и откаты

- Каждое рабочее обновление сохраняйте отдельным коммитом.
- Стабильные версии помечайте тегами: `v0.1`, `v0.2`, `v1.0`.
- Если новая версия сломалась, откат делается через историю GitHub или возврат к прошлому тегу.
